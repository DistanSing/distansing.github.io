What is DistanSing?

DistanSing is a peer-to-peer application that allows musicians to sing and play together over a network with low-latency.

  • Download

How does it work?

DistanSing uses several peer-to-peer networking techniques to connect individuals together directly rather than through a central server. This allows for lower latency and a more natural musical experience.

Peer-to-Peer

Peer-to-peer networking allows for high speed, low-latency audio without the need for a central relay server.

Server - Client

A Central Server is used to set up rooms and manage users. However, the audio data itself is sent peer-to-peer! That way room setup can be reliable and secure, and audio transfer can be fast.
